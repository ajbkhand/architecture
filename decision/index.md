# Decide if you should build an AI agent

Use this page to:

- Decide if an agent is needed  
- Choose the simplest approach  
- Stop early if risks are too high  

---

# Step 1: Define the outcome

You must define a measurable outcome.

✅ Example:
- Reduce case processing time by 20%

---

## Stop if you cannot answer this

❌ If the outcome is unclear, do not proceed  

---

# Step 2: Define the user experience

Ask:

- What does the user do?  
- What does the system do?  
- Where does the human decide?  

---

## Stop if unclear

❌ If the journey is unclear, redesign the use case  

---

# Step 3: Check existing solutions first

You must:

- Use existing tools where possible  
- Start with SaaS  
- Only move to custom build if required  

---

## Delivery ladder

1. SaaS (default)  
2. Low-code  
3. Pro-code  

---

# Step 4: Do you need an agent at all?

Use an agent only if:

- The task is complex or variable  
- It requires reasoning or planning  

---

## Do not use an agent if:

- The process is predictable  
- A workflow can solve it  
- It is simple retrieval  

---

# Step 5: Choose experience type

- **Feature** → small task (preferred start)  
- **Companion** → assist users  
- **Destination** → full workspace  

---

# Step 6: Define autonomy

You must classify the agent:

- Read-only  
- Drafting (human approval)  
- Tool-using (human approval)  
- Actioning (highest risk)  

---

✅ Human approval is required for all consequential actions  

---

# Step 7: Decide tool use

If tools are required:

- Minimise tool access  
- Apply least privilege  
- Log all tool use  

---

⚠️ Tool access is the highest risk area  

---

# Step 8: Output required

You must produce:

- Clear outcome  
- Chosen approach  
- Autonomy level  
- Human control points  
- Evaluation plan  

---

# Do

- ✅ Start simple  
- ✅ Stop early if unclear  
- ✅ Use existing tools first  

---

# Do not

- ❌ Default to agents  
- ❌ Build without controls  
- ❌ Allow unrestricted autonomy  

---

> If the use case is not clear, do not build an agent.

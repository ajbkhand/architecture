# Decision Framework

This is the **mandatory starting point** for any agent proposal.

## Overview

Before building an agent you must:

1. Define a business outcome  
2. Define the user experience  
3. Check existing tools  
4. Decide if an agent is needed  
5. Choose delivery route  
6. Define autonomy and controls  

If you cannot do these → STOP

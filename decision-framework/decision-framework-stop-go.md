# STOP / GO Framework

## Step 1 — Outcome

Define:

“We will improve X by Y for Z”

If unclear → STOP

## Step 2 — User experience

Describe:
- User actions  
- Agent actions  
- Human decisions  

If unclear → STOP

## Step 3 — Existing tools

If already solved → use it

## Step 4 — Agent needed?

If simpler solution works → STOP

## Output required

- Problem statement  
- Experience type  
- Delivery option  
- Autonomy level  
- Control points

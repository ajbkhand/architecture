# Do we need an agent?

Agents are not the default solution.

## Use an agent when

- Tasks are unpredictable  
- Multi-step reasoning is required  
- Tools/APIs are needed  
- You augment human judgement  

## Do NOT use an agent when

- Process is deterministic  
- Inputs/outputs are fixed  
- It is just search or summarisation  
- It removes expert judgement  

## STOP if

- No clear outcome  
- No clear UX  
- Risks cannot be controlled

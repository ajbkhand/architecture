# Standards

Standards define the **minimum required controls** for all agentic systems in Defra.

These are not optional.

## Scope

Standards apply to:

- Agent → Model interactions  
- Agent → Tool interactions (MCP)  
- Agent → Human interactions  
- Security, data, and audit  

## Principles

- Apply least privilege at all times  
- Constrain autonomy and tool access  
- Require human approval for consequential actions  
- Ensure full auditability and traceability

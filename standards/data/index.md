# Data and privacy standards

## Principles

- Lawfulness  
- Data minimisation  
- Transparency  
- Human oversight  

## Requirements

- Conduct DPIA where required  
- Classify data correctly  
- Avoid unnecessary personal data  

## Controls

- Limit data exposure  
- Control data flows  
- Monitor usage  

## Risk

Primary risks:

- Data leakage  
- Unauthorised access  
- Over-collection  

## Outcome

Agents must handle data safely, proportionately, and lawfully.

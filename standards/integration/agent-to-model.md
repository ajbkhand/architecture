# Agent → Model standards

## Requirements

- All model calls must use approved providers  
- All traffic must use HTTPS  
- Route via AI Gateway where available  

## Data controls

- Apply data minimisation  
- Do not send unnecessary personal data  
- Ensure retention policies are defined  

## Logging

Log:

- Input prompts  
- Model responses  
- Metadata (timestamp, user, system)  

## Risk

- Hallucination  
- Data exposure  

Mitigate through:
- Grounding (RAG)
- Evaluation

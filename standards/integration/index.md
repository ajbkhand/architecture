# Integration standards

Agents interact with:

- Models (LLMs)
- Tools (APIs via MCP)
- Humans (HITL)

All integrations must be **governed and auditable**.

## Core requirements

- Use a controlled integration surface (AI Gateway)  
- Enforce authentication and authorisation  
- Log all interactions  
- Apply rate limits and quotas  

## Sections

- Agent to model  
- Agent to tool (MCP)  
- Agent to human

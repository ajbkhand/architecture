# MCP Quadruple Lock

The Quadruple Lock applies layered protection to tool-using agents.

## 1. Tool Surface Lock

- Minimise tools  
- Prefer read-only  
- Limit capabilities  

## 2. Gateway Lock

- All tools accessed via AI Gateway  
- Apply auth, logging, quotas, policies  

## 3. Tenant/Environment Lock

- Apply DLP controls  
- Segregate environments  
- Prevent unsafe connector combinations  

## 4. Evidence Lock

- Require human approval  
- Log all actions  
- Test and evaluate behaviour  

## Outcome

Safe MCP requires:
- Constrained tools  
- Central governance  
- Continuous monitoring

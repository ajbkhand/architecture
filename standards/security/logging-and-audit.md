# Logging and audit

All agent activity must be traceable.

## Mandatory logging

Capture:

- Inputs (prompts, data)  
- Model outputs  
- Tool calls  
- Identity (user/agent)  
- Approvals  

## Requirements

- Logs must be immutable  
- Logs must support audit and incident response  
- Logs must be accessible for review  

## Use cases

- Assurance  
- Security investigations  
- Performance monitoring  

## Principle

No action should occur without traceability.

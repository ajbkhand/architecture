# Security standards

Agentic systems introduce **new risks** due to autonomy and tool use.

## Core principle

> The risk is what the agent can DO, not what it says.

## Scope

- MCP security  
- OWASP-aligned controls  
- Identity and access  
- Logging and audit  

## Mandatory approach

- Constrain tool access  
- Validate all inputs  
- Apply human approval  
- Log everything  
- Test against adversarial scenarios

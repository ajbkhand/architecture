# Tool-Using Agent (MCP)

## Purpose

Interact with systems and APIs.

## Use when

- Calling services  
- Performing actions  

## Controls

- Least privilege  
- Approvals required  
- Log all tool calls  
- Use governed MCP  

## Risks

- Incorrect tool use  
- Excess permissions  
- Prompt injection  

## Principle

Tool use = highest risk, strongest controls required.
